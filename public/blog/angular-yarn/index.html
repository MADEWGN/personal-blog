Menggunakan Yarn pada Angular CLI

Update terakhir: 30 Dec 2017

Angular default-nya menggunakan NPM. Apabila ingin mengganti package manager ke Yarn, cukup ketik perintah ini:

ng set --global packageManager=yarn

Percobaan:

Untuk mengembalikan ke NPM:

ng set --global packageManager=npm

Kenapa Pakai Yarn?

Menurut saya Yarn lebih cepat daripada NPM dan Yarn bisa digunakan Offline.
